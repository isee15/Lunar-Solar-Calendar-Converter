第九章 关于日食的计算

使用方法

为纪念09.7.22长江大日食，在寿星中大量增强日月食等天象计算功能，这年7月之后改名为《寿星天文历》。

显示某日的信息摘要后，如果按住ctrl键不放，再将鼠标移出日历区，即可保持该日信息摘要不变。

寿星万年历中有的地方使用“天文纪年”，有的地方使用“传统纪年”，差别在公元前。“天文纪年”表示公元前的年份用负数表示，传统纪年用BC表示且没有公元0年。本万年历的传统公元前纪年法是前面加一个B或b或*字符。例如输入公元前212年的方法是B212或*212或-211。

日月升降使用本地时间，气朔时刻使用北京时间。

地标数据库应与时区相匹配(相差10小时以内问题不大)，否则可能造成太阳升降日期出错1天。如果自行换的地标经纬度超出压缩算法所许可的范围(默认为中国范围)，则需要修改压缩算法。
